# Iron Snout Fun - 项目简介

## 项目概述
Iron Snout Fun 是一个游戏网站项目，需要集成 Google AdSense 广告系统来实现盈利。

## 核心需求
- 集成 Google AdSense 广告代码
- 客户端 ID: ca-pub-3219924658522446
- 需要在网站中嵌入广告展示

## 技术要求
- 使用 Google AdSense JavaScript API
- 异步加载广告脚本
- 跨域安全配置

## 项目目标
- 创建一个功能完整的游戏网站
- 集成广告系统实现收益
- 提供良好的用户体验
